### üè† Tabela **ENDERECO**

    ESTADO CHAR(2) NOT NULL,

> **Observa√ß√£o:** A tabela *ENDERECO* exige o cadastro de apenas um endere√ßo por cliente.

### üìû Tabela **TELEFONE**
